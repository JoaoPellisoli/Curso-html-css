Configurações que vão no font-weigth:
lighter, normal, bold e bolder.

Palavras parecidas:
weight = peso
width =  largura
height = altura

dependendo da fonte algumas configurações das CSS não funcionam.

shorthand font facilta a configuração das CSS:(NÃO EXISTE SHORTHAND PRA TUDO)
Normal:
h1 {
            font-family: 'Work Sans', 'sans-serif';
            font-weight: bolder;
            font-size: 3em;
            font-style: italic;
        }

Shorthand font:
Nesta ordem: font-style > font-weigth > font-size > font-family.
h1{            
	   font: italic bolder 3em sans-serif;
	}
