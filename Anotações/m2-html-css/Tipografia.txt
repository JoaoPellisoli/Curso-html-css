Tipografia:
Tipografia é o estudo dos tipos.

Anatomia do tipo:
Todas as fontes são criadas apartir da altura da letra x minúscula. Essa médida e chamada de altura-x
Altura das maiusculas são chamadas de alturar das maiúsculas, são um pouco maiores que altura-x.
a letras que são maiores que a altura-x, são as ascendentes.
as que tem cauda como a letra Q são descendente.
Somando todas as altura se chama corpo.

Glifos, Fontes , familia:
Glifos são as letras.
Todos os glifos de uma determinada familia são chamados de fontes.
Família tipográfica: É o tamaho da fonte, são cinco os tamanhos, light, normal, seminegrito, negrito, extranegrito,  
nem todas as fontes tem todas essas variações, as vezez nem variações tem.

Categorias de Fontes:
Serifadas: Letras com serifas.
Sans-serif: Letras sem serifas.
Monoespaçadas: Letras que tem o mensmo tamanho.
Handwriting: Letras que simula escritura a mão.
Display: Letras comemorativas, não se preucupa com as regras acima.
