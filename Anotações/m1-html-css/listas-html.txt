Listas ordenadas 
lista de um o passo-a-passo e a ordem do itens são importante.
para criar e usado <ol>(ordered list) ela delimita a lista e. <li>(list item) para indentificar cada item da lista
a tag <ol> tem uma parêmetro type para configurar a lista, exemplo:
1 - Ex: 1, 2, 3, 4,...
A - Ex:A, B, C, D,...
a - Ex:a, b, c, d,...
I - Ex:I, II, III, IV,...
i - Ex:i, ii, iii ,iv,...
Pode ser usado tambem o parâmetro start para indicar o inicio da contagem.
Ex: 1 -<ol type="I" start="5"> ai gerar os item como V, VI, VII,...

Lista não ordenadas
Lista que não precissam estár na ordem certa. Ela e ussada para organizar os itens, são listas com marcadores.
Para criar e usado <ul> (unordered list) para delimitar a lista é <li> para criar cada um dos seus items internos.
o marcador pradão e bolinha preta toda preenchida. mas da para configurar com type dentro da <ul> com:
disc-Pradrao
circle-Uma bolinha sem preenchimento.
square - Um Quadrado pintado.

Listas de definições
É como se fosse um dicionario, temosos termos e as suas descrições. não tem demarcadores.
Toda a lista de definições está dentro de uma tag <dl> (definition list). Cada termo e um <dt> (definition term)
é cada descricrição e um <dd> (definition description).
Ex:
<dl>
<dt>HTML</dt>
<dd>Linhguagem de marcação utilizada para criar conteúdo de sites.</dd>
<dt>CSS</dt>
<dd>                 Descrição                                    </dd>
</dl>
