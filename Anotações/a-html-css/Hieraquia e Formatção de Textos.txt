Hieraquia de Textos
O titlulo principal começa com a tag <h1>
para o sub-titulo usamos <h2> e a assim sucessivamente.

Formatação de textos
<strong> usado para colocar o texto em negrito
<em> usado para dar ênfase ao texto
<mark> usado para dar ênfase com uma marca textos
<small> para textos pequenos.
<del> significa que o texto está ali mas pode ser ignorado.
<ins> e o oposto da tag del o texto está ali para ser lido e devemos prestar atenção nele.
<sub> para colocar um numero pequeno embaixo.
<sup> para colocar um numero pequeno em cima como x¹, por exemplo.
<code> para colocarmos exemplos de códigos usamos, elas ficaram mono-espaçadas
<pre> utilizamos junto com o <code> para o texto ficar exatamente como foi escrito, incluindo quebras de linhas espaços e tabulações
<q> quando queremos uma citação dendro de um parágrafo. Ela coloca citações.
<blockquote> para citações mais longas, que tem um parágrafo só para si. podemos por um paramentro chamado CITE dentro da tag para colocar um link para o texto original.
<abbr> usado quando por abreviações, exemplo:HTML quando colocar o mouse em cima aparecera Hyper Text Markup Language.
<bdo> Para inverter a ordem de um título ou parágrafo, usando o parametro DIR para indicar uma das duas direções possíveis
rtl = da direita para esquerda
ltr = da esqueda pra direita
