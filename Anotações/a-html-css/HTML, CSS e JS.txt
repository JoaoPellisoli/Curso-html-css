HTML: Foco em conteúdo, pode ser textos,imagens,vídeos,tabelas etc.
CSS: desing como cores, sombras,tamanhos, posicionamento etc.
JS(JavaScript): interações,como menus, animações,popups,validações

Exemplos:
		       conteúdo
(abertura da tag)<h1>Exemplo de título</h1>(Fechamento da tag)
<p>Exemplo de parágrafo</p>
<img src= "foto.png" alt="Exemplo de foto">
		   parâmetro |  valor

Estilo em CSS:
(seletor)h1{ font-family: Arial;
	     font-size: 20pt; 
	     (propriedade)color:blue(valor);	
	   }(tudo aqui dentro são declarações)

Estrutura básica de documento HTML
<DOCTYPE html>
<html lang = "pt-br">
<head> (configurações do site são feitas no head)
<meta charset="UTF-8">(Aceita todos tipos de caracteres)
<meta name="viewport"
content="width=device-width,initial-scale=1.0">(serve para o site entrar já ocupando 100% da tela)
<title>Document</title>(Aqui vai o título do site)
</head>
<body> (Corpo do site)
<h1>Olá, Mundo!</h1>
</body>

</html>
